# تقرير دورة حياة التخزين

أوامر cleanup للمعاينات والفشل والمؤقتات جافة افتراضياً. تعرض رمز تأكيد
خاصاً بالهدف، وعند التنفيذ تنقل الملفات إلى سلة خارجية قابلة للاسترجاع.
لا تحذف بيانات المصدر ولا manifests. يسجل تقرير التدقيق المسار والحجم
وSHA-256.

`archive-run` ينشئ ZIP جديداً من دون استبدال، ويضيف manifest v1 لكل عضو.
`verify-archive` يفحص traversal والروابط والحجم وعدد الأعضاء وبصمات الملفات.
