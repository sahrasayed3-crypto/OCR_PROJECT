# العوائق الخارجية

- اختيار وإعداد مزود OIDC، client IDs، claims وtenant policy.
- شهادة TLS وreverse proxy موثوق وقائمة proxy headers.
- Redis production ACL/TLS/network policy.
- مخزن أسرار مُدار ودورة rotation.
- مراجعة قانونية لرخص datasets والنماذج والاستخدام التجاري.
- سياسات retention/deletion والنسخ الاحتياطي وRPO/RTO.
- توقيع wheel/sdist وprovenance وسياسة نشر CI.
- اختبار تحميل إنتاجي ومراقبة/تنبيه/استجابة حوادث.

هذه ليست صلاحيات يمكن اختلاقها من الشيفرة، ولذلك تبقى بوابة النشر العام محجوبة.
