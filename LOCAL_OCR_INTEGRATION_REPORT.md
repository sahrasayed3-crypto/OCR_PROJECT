# تقرير تكامل OCR المحلي

المحولات: mock، command line، local HTTP، OpenAI-compatible local، Transformers VLM وQwen-VL-compatible.

الافتراضي آمن: معطل، `local_files_only=true`، لا تنزيل تلقائي، `trust_remote_code=false`، حدود للصورة والتوكن والمهلة والدفعة، ومسار نموذج داخل الحدود المسموحة.

runtime يحافظ على النص الرقمي، ويبقي المسح `pending_ocr_model` عند التعطيل. عند التفعيل يستدعي المحول ويمنع نجاح النص الفارغ ويسجل هوية النموذج والمراجعة والزمن والجودة ويحوّل الفشل/الجودة المنخفضة للمراجعة.

الاختبار والقبول استخدما mock فقط؛ الحالة النهائية `local_model_ocr` وDOCX صالح. النموذج الحقيقي عائق خارجي ومعطل افتراضيًا.
