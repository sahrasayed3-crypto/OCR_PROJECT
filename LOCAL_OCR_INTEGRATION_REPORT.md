# تقرير تكامل OCR المحلي

المحولات الآمنة موجودة لـ:

- HTTP محلي؛
- endpoint محلي متوافق مع OpenAI؛
- محرك سطر أوامر بمسار مطلق و`ـshell=False`؛
- Transformers/Qwen-VL من ملفات محلية فقط؛
- محرك وهمي اصطناعي للاختبار وببوابة صريحة.

ربط التشغيل مكتمل خلف `CLOUDA_LOCAL_OCR_ENABLED`. يبقى النص الرقمي أولاً،
وتظل الصفحات الممسوحة pending عند التعطيل. عند التفعيل يسجل النموذج والمراجعة
والزمن والثقة، ولا يقبل نصاً فارغاً. النموذج الحقيقي غير مختار ومعطل.
