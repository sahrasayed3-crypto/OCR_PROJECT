# افتراضات الأمن

- إعدادات البيئة وملفات الخدمة سلطة إدارية وليست مدخل مستخدم.
- Redis وواجهة العامل غير قابلين للوصول من الإنترنت.
- مفتاح العامل طويل وعشوائي وقابل للتدوير ويُخزن خارج Git.
- مستخدمو CLI المحليون لديهم أصلًا صلاحية OS على الجذور المهيأة.
- catalog المتتبع يحتاج مراجعة مالك البيانات؛ manifest ليس مصدر ترخيص سلطويًا.
- `runtime://` و`dataset://` و`artifact://` و`model://` و`cache://` جذور منفصلة.
- لا يُسمح لنموذج محلي بتنزيل ملفات أو تنفيذ Python بعيد.
- OCR/document content بيانات فقط ولا يملك سلطة أو tools.
- الخدمة متعددة المستخدمين غير مدعومة حتى دمج هوية وملكية tenant.
- TLS ينتهي عند proxy موثوق؛ HSTS ليس مفعلًا تلقائيًا.
- أدوات الأمن غير المتاحة تُسجل صراحة ولا يُدعى تشغيلها.
