# تقرير تنفيذ التصيير

التصيير مكتمل لـ PDF وPNG وJPEG وTIFF متعدد الصفحات وWEBP. يدعم نطاق
الصفحات، وDPI، والحد الأقصى للأبعاد والبكسلات، والألوان/الرمادي/الثنائي،
وتصحيح EXIF، وتسوية alpha، والأسماء Unicode. تعالج الصفحات واحدة تلو الأخرى.

يسجل manifest v1 بصمات SHA-256 للمصدر والناتج، والأبعاد، والمحرك وإصداره،
وهاش الإعداد، والحالة والخطأ. الكتابة ذرية ولا تستبدل ملفات قائمة. يدعم
dry-run والاستئناف والتحقق.
