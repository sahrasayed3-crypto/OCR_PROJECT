# العناصر غير المكتملة

## عوائق خارجية

1. موافقة قانونية نهائية على كود Project B.
2. موافقة البيانات المعلقة أو البحثية للاستخدام التجاري.
3. مزود OIDC/SSO إنتاجي وأسراره.
4. شهادة TLS ونطاق نشر عام.
5. اختيار نموذج OCR نهائي، وترخيصه، وأوزانه، وتقييمه.
6. GPU وحساب سحابي مدفوع.

## تحسينات مستقبلية غير مانعة

- التنفيذ المتوازي للتشويه: يقبل CLI عدد العمال ويتحقق من حدوده، لكن التنفيذ
  الحالي تسلسلي وحتمي للحفاظ على البصمات وكتابة manifest الذرية.
- القياس الدقيق للذاكرة وتوسّع CPU يحتاج منصة benchmark مستقرة.
- perceptual hashing موجود كواجهة حقن؛ لا توجد مكتبة اختيارية مفروضة.

هذه العناصر لا تمنع التشغيل المحلي الآمن أو معايير الإكمال الأساسية.
