# تقرير الأداء والموارد

- قبول شامل: 30 صورة مشوهة مع التصيير والتحقق والمعاينة والتصدير وOCR
  التجريبي خلال 13.72 ثانية؛ هذا قياس workflow وليس benchmark معزولاً.
- المعدل الشامل التقريبي: 131 صورة/دقيقة.
- الحدود: 100 صفحة افتراضياً، 8 عمال كحد إدخال، 40 مليون بكسل، حد bytes
  للناتج، وحد أدنى لمساحة القرص، ومعالجة صفحة بصفحة.
- الإلغاء/الاستئناف: انقطاع مقصود واستئناف ناجح.
- المخرجات الجزئية: أسماء مؤقتة وكتابة ذرية.
- التنفيذ الحالي: تسلسلي حتمي؛ worker count المتوازي تحسين مستقبلي حتى لا
  يضعف ترتيب manifest أو قابلية الإعادة.
- الذاكرة القصوى لم تُقاس بأداة مستقلة، لذلك لا توجد مطالبة رقمية.
