# المخاطر المتبقية

لا توجد ثغرة برمجية مؤكدة غير محلولة ضمن بوابة المنع المحددة. المخاطر المتبقية:

1. OIDC/SSO إنتاجي غير مهيأ؛ النظام يفشل مغلقًا ويجب أن يبقى خاصًا.
2. TLS وtrusted proxy وWAF/rate limiting الموزع مسؤولية طبقة النشر الخارجية.
3. Redis المشترك يحتاج ACL وTLS وعزل writers.
4. موافقات تراخيص datasets/models وDPA/retention قرارات قانونية وتشغيلية.
5. لا توقيع artifacts أو SLSA provenance بعد.
6. Bandit يبقي 33 تحذيرًا سياقيًا: SQL templates ذات allowlists، URL محلي متحقق منه، وTransformers local-only؛ يلزم إعادة مراجعتها عند تغير الحدود.
7. التغطية 75.12% وليست إثباتًا كاملًا؛ المسارات الاختيارية الخاصة بالسحابة/GPU تحتاج بيئة اعتماد منفصلة.
