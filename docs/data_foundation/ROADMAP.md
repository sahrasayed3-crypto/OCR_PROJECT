# Roadmap

1. Foundation: structure, configs, schemas, policies, interfaces, tests, docs.
2. Ingestion: accept clean PDFs, DOCX, images, text, PAGE XML, ALTO XML, and layout JSON.
3. Rendering: render clean pages reproducibly.
4. Distortion: implement safe page and layout-aware visual operations.
5. Validation: reject unusable outputs before they enter experiments.
6. Evaluation: run OCR engines and calculate CER/WER.
7. Scaling: add resumable local batches.
8. AWS: add explicit, reviewed cloud execution.
