# Model selection

No final model is selected. The registry contains a disabled placeholder with
an unresolved revision and license review requirement. A candidate must record
an exact base revision, tokenizer and processor revisions, license, architecture,
parameter counts, supported images, training method, dataset manifests,
evaluation policy, commercial status, attribution, and deployment decision.

Qwen-VL-compatible providers may be implemented through adapters, but no model
family is hard-coded as the target.
